# Le tue fotografie

Colloca in questa cartella file **JPG** con questi nomi esatti. All'avvio, ogni fotografia
presente sostituisce automaticamente la texture procedurale corrispondente. Nessuna modifica
al codice è necessaria.

| File | Superficie | Suggerimento di ripresa |
|---|---|---|
| `pietra_ciclopica.jpg` | Mura ciclopiche | Paramento murario frontale, luce diffusa, ~2×2 m di muro |
| `conglomerato.jpg` | Porta dei Leoni (stipiti, architrave) | Dettaglio ravvicinato del conglomerato («pietra a mandorle») |
| `lastra.jpg` | Lastre e stele del Circolo A, pavimenti | Superficie di una lastra calcarea liscia |
| `intonaco.jpg` | Pareti intonacate di case e palazzo | Intonaco chiaro, anche moderno purché neutro |
| `affresco.jpg` | Fascia dipinta del megaron (1024×256 circa) | Fascia orizzontale; verrà ripetuta 3 volte in larghezza |
| `legno.jpg` | Porte, travi, colonne di ulivi | Venatura verticale |
| `terra.jpg` | Terreno della collina | Terra battuta con erba secca, ripresa dall'alto |
| `roccia.jpg` | Monti Profitis Ilias e Zara | Parete rocciosa a distanza |

Indicazioni tecniche:
- dimensioni consigliate: 1024×1024 (l'affresco 1024×256);
- le immagini vengono ripetute (tiling): preferisci inquadrature uniformi, senza ombre nette ai bordi;
- lo stato di ogni slot (procedurale / fotografia) è visibile nel menu dell'app.
